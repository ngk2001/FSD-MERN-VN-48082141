### Project 2: FlightHub - Flight Booking Application
A comprehensive full-stack flight booking application built with MongoDB, Express.js, React, and Node.js.

**Features:**
- User authentication and authorization
- Flight search and booking system
- Real-time flight availability
- Admin dashboard for flight management
- Booking management (Cancel/Reschedule)
- User profile management

**Tech Stack:**
- **Frontend:** React, React Router, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT
